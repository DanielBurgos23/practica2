Navegadores Web

Un navegador web es un software, aplicacion o programa que permite el acceso a la web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad basica de un navegador web es permitir la visualizacion de documentos de texto.

Ejemplos de navegador Web

Safari: es un navegador web de codigo cerrado desarrollado por Apple.

Mozilla Firefox es un navegador web libre y de codigo abierto desarrollado por linux

Google Chrome es un navegador web de codigo cerrado desarrollado por Google
